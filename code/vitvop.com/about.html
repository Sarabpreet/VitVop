Voice Oriented Programming

Project VitVop

Project Aim :

To help visually impaired and disabled people, to do computer programming through voice recognition.

Feature:

  • Existing software's are based on native platform. But the new application is a web based application therefore platform independent.
  • Technology used is Html5, JavaScript and Foundation.
  • Automatic generation of code ,output and error via voice/speech.
  • Mobile friendly web based application.
  • It is an open source project(source code is on github).

Future Implimentation

  • Implementation in different programming language such as Java, Python, Ruby, C#, C++ etc.
  • Inbuilt Complier Engine to convert the text into code output.
