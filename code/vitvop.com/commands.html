Voice Oriented Programming

Supported Voice Commands

  • "hash" -> #
  • "dot" -> .
  • "open c" -> {
  • "close c" -> }
  • "open p" -> (
  • "close p" -> )
  • "open b"-> [
  • "close b" -> ]
  • "next line" -> cursor goes to the new line
  • "space" -> generates an extra white space
  • "clear" -> clears the output area
  • "hello" -> Prints the default hello program in C language.

Yes, we built it for a change.
